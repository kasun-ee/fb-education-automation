This project is a fully automated Facebook education content engine
designed for Sinhala educational posts.

## Daily Content Types
1. Country of the day
2. History & technology
3. Image difference challenge
4. New technology explanation
5. Quiz post

## Technologies Used
- Meta Facebook Graph API
- Gemini API (content generation)
- GitHub (version control)
- Python (automation)
- Google Drive (media storage)

## Goal
To publish 5 high-quality Sinhala educational posts daily
with minimal human intervention and continuous self-improvement.
